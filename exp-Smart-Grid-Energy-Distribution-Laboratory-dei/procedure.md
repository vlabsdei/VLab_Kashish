## Step 1: Launch the Simulation

1. Open the Smart Grid Energy Distribution Laboratory simulation.
2. Observe the major components of the smart grid:

   * Solar Power Generation Unit
   * Wind Power Generation Unit
   * Utility Grid
   * Battery Storage System
   * Grid Controller
   * Distribution Network
   * Residential Load
   * Commercial Load
   * Industrial Load


## Step 2: Set Generation Parameters

1. Adjust the Solar Generation slider to select the desired solar power output.
2. Adjust the Wind Generation slider to select the desired wind power output.
3. Observe the total renewable energy generation.


## Step 3: Set Load Demand Parameters

1. Adjust the Residential Load slider.
2. Adjust the Commercial Load slider.
3. Adjust the Industrial Load slider.
4. Observe the change in total power demand.


## Step 4: Select Grid Operating Mode

1. Choose the desired grid operating mode:

   * Grid Connected Mode
   * Island Mode (Standalone Operation)

2. Observe how power flow changes according to the selected mode.


## Step 5: Apply Operating Scenarios

Select any preset scenario:

* Peak Load
* Storm Event
* Night Time
* Eco Mode

Observe the corresponding changes in:

* Power generation
* Load demand
* Battery operation
* Grid condition


## Step 6: Start the Simulation

1. Click the Start Grid Simulation button.
2. Observe the animated power flow between different grid components.
3. Monitor energy transfer from generation sources to consumers.


## Step 7: Observe Battery Storage Operation

1. Monitor the battery charge level.

2. Observe charging when:

   * Total Generation > Total Demand

3. Observe discharging when:

   * Total Demand > Total Generation

4. Record battery percentage changes.


## Step 8: Monitor Grid Outputs

Observe and record:

* Generated Power (kW)
* Total Demand (kW)
* Power Surplus/Deficit (kW)
* Grid Efficiency (%)
* Renewable Share (%)
* Grid Condition


## Step 9: Analyze SCADA Graph

1. Observe the real-time SCADA Analytics graph.

2. Compare:

   * Generation Curve
   * Demand Curve

3. Analyze system performance under different operating conditions.


## Step 10: Study Different Cases

### Case A: Surplus Generation

1. Set generation greater than demand.
2. Start the simulation.
3. Observe:

   * Battery charging
   * Grid exporting power
   * High efficiency

### Case B: Balanced Operation

1. Adjust generation approximately equal to demand.
2. Observe stable grid operation.

### Case C: Power Deficit

1. Increase demand above generation.
2. Observe:

   * Battery discharging
   * Utility grid support
   * Reduced system efficiency


## Step 11: Record Observations

For each trial, record:

* Solar Generation
* Wind Generation
* Residential Load
* Commercial Load
* Industrial Load
* Total Generation
* Total Demand
* Battery Status
* Grid Efficiency
* Renewable Energy Share


## Step 12: Reset the Experiment

1. Click the Reset Parameters button.
2. Restore all parameters to their default values.
3. Repeat the experiment with different input combinations.


## Result

The Smart Grid Energy Distribution system was successfully simulated. The effect of varying renewable generation and load demand on power balance, battery storage operation, grid efficiency, renewable energy contribution, and overall grid performance was observed and analyzed.
